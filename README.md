# 🚀 Multimodal Clickbait Detection System

## Features
- Text-based clickbait detection
- Thumbnail OCR analysis
- Video speech analysis
- Hybrid decision system
- Explainable AI output

## Tech Stack
- Python (Flask)
- TF-IDF + Logistic Regression
- EasyOCR
- SpeechRecognition
- FFmpeg

## How to Run

```bash
pip install -r requirements.txt
python app.py
